Query Syntax Guide

Search and filter cards using a powerful text-based query language. Combine multiple filters to find exactly what you need.

Basic Search

elsa Search card names (fuzzy match)

Ink Colors

i:y or i:amber Filter by ink color
Color shortcuts: y yellow gold = Amber p purple = Amethyst e g green = Emerald r red = Ruby b u blue = Sapphire s steel silver = Steel
i:yp Multiple colors (yellow OR purple)
i!gs ONLY green/steel (excludes green+other or steel+other)
i:ym Multicolor cards containing yellow
m or multicolor All multicolor cards

Card Properties

c:4 or c=4 Cost equals 4
c>=4 c<=7 Cost comparisons (>=, <=, >, <, !=)
s>=5 Strength 5 or higher
w>=4 Willpower 4 or higher
l>=2 Lore 2 or higher

Card Types

t:character Characters only
t:action t:item t:location t:song Other card types

Text & Attributes

o:draw Text contains "draw" (ability search)
k:hero Characteristics (Hero, Villain, Storyborn, etc.)
r:rare Rarity (common, uncommon, rare, super, legendary)

Format

Format is controlled by the Core/Infinity toggle at the top of the cards page. You don't need to include format in your queries.

Inkwell

inkable Can be played as ink
noninkable Cannot be played as ink

Negation

-t:song Exclude songs
-inkable Exclude inkable cards

Sorting Results

sort:cost or s:cost Sort by cost (ascending)
sort:-cost or s:-cost Sort by cost (descending) - prefix with - to reverse
sort:color Sort by ink color (Amber → Steel)
sort:type Sort by card type
sort:set Sort by set and collector number
sort:lore Sort by lore (descending by default)
sort:str sort:will Sort by strength or willpower (descending by default)
sort:-name Reverse alphabetical order (Z → A)
Note: Default sort is by name. Add - prefix to any sort field to reverse the order (e.g., sort:-set for newest sets first).

Combining Filters

i:y c>=4 t:character sort:cost Yellow characters costing 4+, sorted by cost
i:yp l>=2 inkable sort:-lore Yellow or purple, 2+ lore, inkable, sorted by lore (highest first)
o:draw -t:song sort:-set Cards with "draw", excluding songs, newest sets first

Tips

  • Use the Core/Infinity toggle at the top of the cards page to control format
  • Filters are separated by spaces
  • Multiple filters of same type = OR (e.g., i:y i:p = yellow OR purple)
  • Different filter types = AND (e.g., i:y c>=4 = yellow AND cost 4+)
  • Press Space or Enter to apply filters
  • Queries persist in the URL for easy sharing
